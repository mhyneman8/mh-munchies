# MH Munchies

A Next.js restaurant discovery and filtering application.

## Prerequisites

- Node.js (v18 or higher)
- pnpm (v10.25.0 or higher) - This project uses pnpm as the package manager

## Getting Started

### Installation

1. Clone the repository:
```bash
git clone <https://github.com/mhyneman8/mh-munchies>
cd mh-munchies
```

2. Install dependencies:
```bash
pnpm install
```

### Development

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

The page will auto-update as you edit files.

### Available Scripts

- `pnpm dev` - Start the development server
- `pnpm build` - Build the application for production
- `pnpm start` - Start the production server (requires `pnpm build` first)
- `pnpm lint` - Run ESLint to check for code issues

## Tech Stack

- **Next.js 16** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **CVA** - Component variant API for styling
